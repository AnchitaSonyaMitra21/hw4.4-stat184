# Project and Repository Plan

## 1. Project Plan (HW 4.3)

### Goals
- Analyze how calcium levels change over time
- Create clear visualizations to show trends
- Present results in a well-structured report

### Needs
- Dataset: calcium.csv
- R with libraries (ggplot2, dplyr)
- Quarto (.qmd file) for analysis and report

### Steps
1. Load and clean the dataset
2. Reshape data if needed for plotting
3. Create line plots to show trends over time
4. Analyze and interpret the results
5. Write explanations and conclusions
6. Render the final report as a PDF
